# Titanic Exploratory Data Analysis

## Objective
To analyze the Titanic dataset and extract patterns using statistical and visual methods.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

## Project Structure
- titanic_eda.py
- train.csv
- plots/
- Titanic_EDA_Report.pdf

## Key Findings
- Females had higher survival rate.
- First class passengers survived more.
- Missing values found in Age and Cabin.
